# 公共组件

“公共组件”用来查看别人已经开放给你使用的组件。  
它和“我的组件”最大的区别是：这里看到的，不一定是你自己创建的，而是别人授权出来让你能复用的组件。

只要别人把组件权限设置成下面这些范围之一，你就可能在这里看到：

- `所有人可见`
- `企业可见`
- `指定你可见`

![公共组件](/agilebi/public-component-list.png)

## 这页重点

- 组件名称：快速判断组件用途
- 类型：区分它是普通组件还是组合组件
- 描述：帮助理解组件适合放在哪类页面里
- 预览：先确认组件效果，再决定要不要复用

这里更像是共享组件库，不是自己做组件的入口。

## 为什么这里能看到别人的组件

根本逻辑和公共模板一样：  
不是因为组件天然公共，而是因为**对方把权限开放给你了**。

常见情况有三种：

1. 对方设置 `所有人可见`
2. 对方设置 `企业可见`
3. 对方指定当前账号可见

只要满足其中一种，这个组件就可能出现在你的“公共组件”里。

## 这里通常怎么用

“公共组件”更适合这些场景：

- 团队里已经有人做过成熟组件
- 你不想从头重复搭一遍
- 希望直接借用一个表格块、筛选块、时间控件或组合组件

通常流程会是：

1. 先在公共组件里搜索
2. 看名称和描述判断是否适合
3. 预览一下组件效果
4. 再放进自己的页面或项目里使用

所以这页的本质，是共享复用，不是组件制作。
