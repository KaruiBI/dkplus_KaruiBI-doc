# 公共模板

“公共模板”不是系统里所有模板的总库，而是**当前账号有权限看到的公共模板集合**。  
只要别人把模板权限设置成下面这些范围之一，你就可能在这里看到：

- `所有人可见`
- `企业可见`
- `指定你可见`

所以这页最关键的理解不是“公共 = 全开放”，而是“公共 = 对当前账号开放的共享模板”。

## 这页怎么看

- 缩略图：先快速判断模板大概长什么样
- 名称：识别模板用途
- 可见范围：判断为什么你能看到这份模板
- 操作列：通常可以直接 `预览` 或 `复制模板到我的项目`

这里更适合做模板借用，而不是模板沉淀。  
沉淀模板一般在“我的模板”，借用别人开放出来的模板一般在这里。

## 为什么这里能看到别人的模板

核心原因就是对方在模板权限里做了开放。  
只要满足下面任意一种，你就可能在“公共模板”看到它：

1. 对方设置了 `所有人可见`
2. 对方设置了 `企业可见`，并且你在同一企业范围内
3. 对方明确把模板授权给你可见

如果对方设的是“仅自己可见”，那你在公共模板里通常是看不到的。

## 这里通常怎么用

最常见的用法是：

- 先看公共模板列表
- 判断这份模板是否适合当前项目
- 通过 `预览` 看结构是否合适
- 再通过 `复制模板到我的项目` 拿回去继续改

所以“公共模板”更像共享模板市场，而不是编辑入口。
