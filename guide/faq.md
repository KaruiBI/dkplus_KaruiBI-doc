# 常见问题

## 为什么组件标签不显示

通常是下面几个原因：

- 没有执行 `app.use(dkPlus)`
- 没有引入 `@dk-plus/theme-chalk`
- 当前项目不是 Vue 3
- 组件名称写错

## 为什么样式看起来不对

先检查：

1. 样式包是否成功安装
2. 样式是否在入口文件里最先引入
3. 项目里是否有全局样式覆盖了组件默认样式

## 官网后续该补什么

如果你准备把这里做成正式官网，建议优先补这几类内容：

- 首页产品截图
- 核心组件总览
- 每个组件的示例与 API
- 版本更新日志
- 部署地址与演示入口

## 文档站如何启动

```bash
npm install
npm run docs:dev
```

## 文档站如何打包

```bash
npm run docs:build
```
