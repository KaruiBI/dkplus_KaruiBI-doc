# 文档导览

如果你的目标是“尽快把 `dkPlus` 用起来”，建议按照下面顺序阅读：

1. 先看 [为什么选择 dkPlus](/guide/intro)，确认它是否适合你的项目。
2. 再看 [安装](/guide/install)，完成依赖接入。
3. 接着看 [快速开始](/guide/quick-start)，跑通第一个页面。
4. 如果你要给业务同学提供系统使用说明，直接看 [AgileBI V1.0 操作手册](/guide/agilebi-operation)。
4. 最后看 [组件接入说明](/components/)，开始按模块补具体页面。

## 这套文档面向谁

- 第一次接触 `dkPlus` 的前端同学
- 只想先把页面跑起来的业务开发
- 需要给客户或团队成员展示产品能力的人

## 文档设计原则

- 不让用户先读源码
- 不让用户先理解仓库全貌
- 优先解决“怎么装、怎么用、为什么不生效”
