# 组件接入说明

这里先作为 `dkPlus` 组件文档的总入口，后面你可以继续把每个组件拆成单独页面。

## 推荐的扩展结构

```text
components/
  index.md
  button.md
  input.md
  form.md
  table.md
  chart.md
```

## 每个组件页建议包含什么

1. 组件用途
2. 基础示例
3. 常用属性
4. 事件说明
5. 业务场景示例
6. 常见错误

## 推荐先补的组件

- Button
- Input
- Form
- Dialog
- Table
- Charts

## 建议写法

对用户来说，最有用的不是一大堆参数表，而是：

- 先给一个能直接复制的例子
- 再解释关键参数
- 最后补 API 表格
